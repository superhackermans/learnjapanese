제6장조수사 -- 세는 법의 형태 (助数詞)

기억 기법 스포트라이트: 청킹 + 음성 연상

조수사는 수십 개가 넘지만, 음변화 패턴으로 그룹화(청킹)하면 학습이 훨씬 빨라집니다. 예를 들어, 本(ほん), 杯(はい), 匹(ひき)는 모두 h행으로 시작하므로 같은 음변화 규칙이 적용됩니다. 이 패턴을 하나의 덩어리(chunk)로 묶고, 각 변화에 재미있는 음성 연상을 붙이면 개별 암기보다 3배 이상 효율적으로 기억할 수 있습니다.

핵심 전략: "1, 6, 8, 10은 빵빵 터진다 (h → p)" / "3과 ?는 부드럽게 변한다 (h → b)"

궁전 배치: 부엌

기억의 궁전에서 이 장의 무대는 부엌입니다. 각 카운터 카테고리를 부엌의 특정 위치에 배치하세요.

  • 냉장고 문 — つ (만능 카운터): 냉장고 안에 이것저것 다 들어 있습니다.
  • 식탁 — 人(にん): 식탁에 사람들이 앉아 있습니다.
  • 과일 바구니 — 個(こ): 사과, 오렌지 등 작은 물건이 담겨 있습니다.
  • 칼꽂이 — 本(ほん): 길고 가는 칼이 꽂혀 있습니다.
  • 도마 위 — 枚(まい): 평평하게 썬 햄 조각들이 놓여 있습니다.
  • 찬장 — 杯(はい): 컵과 잔이 줄지어 있습니다.
  • 창문 옆 — 匹(ひき) / 頭(とう): 창밖에 동물이 보입니다.
  • 가전 코너 — 台(だい): 전자레인지, 믹서기 등 기계가 놓여 있습니다.
  • 선반 위 책꽂이 — 冊(さつ): 요리책이 꽂혀 있습니다.
  • 현관 신발장 — 足(そく): 신발이 정리되어 있습니다.

복습 확인: 제2장~제5장

이전 장에서 배운 내용을 떠올려 보세요. 아래 항목을 기억하고 있나요?

  • おはようございます의 뜻은? (안녕하세요 - 아침 인사)
  • 히라가나 あ행의 5글자를 순서대로 쓸 수 있나요?
  • 가타카나 カ행의 5글자를 순서대로 쓸 수 있나요?
  • すみません은 어떤 상황에서 사용하나요? (실례합니다 / 죄송합니다)
  • 장음과 단음의 차이가 의미를 바꾸는 예를 하나 들 수 있나요?

6A: 서론 — 왜 조수사가 존재하는가

영어에서는 "three apples", "five books"처럼 숫자 + 명사만으로 셀 수 있습니다. 하지만 일본어에서는 반드시 숫자 + 조수사(카운터)를 함께 사용해야 합니다. りんご三個(사과 세 개)처럼 말이죠.

이것은 일본어만의 특이한 체계가 아닙니다. 한국어, 중국어, 태국어, 말레이어 등 아시아 여러 언어에 공통적으로 존재하는 문법 구조입니다.

한국어의 장점

한국어 화자에게 좋은 소식입니다 — 한국어도 조수사 체계를 사용합니다!

한국어의 주요 조수사를 떠올려 보세요:

  • 개 — 일반 사물 (사과 세 개)
  • 명 / 사람 — 사람 (학생 다섯 명)
  • 마리 — 동물 (고양이 두 마리)
  • 잔 — 음료 (커피 한 잔)
  • 병 — 병에 든 것 (맥주 세 병)
  • 장 — 종이, 평평한 것 (종이 열 장)
  • 권 — 책 (책 두 권)
  • 대 — 기계, 차량 (차 한 대)
  • 벌 — 옷 (양복 한 벌)
  • 켤레 — 신발 (신발 두 켤레)

이 개념 자체가 익숙하므로, 일본어 조수사를 배울 때 "어떤 카테고리에 어떤 카운터를 쓰는지"만 대응시키면 됩니다. 한국어에 없는 새로운 개념을 배우는 것이 아니라, 이미 가진 지식에 새 라벨을 붙이는 것입니다.

일본어 조수사의 핵심 난이도는 두 가지입니다:

  1. 숫자와 결합할 때 발음이 변하는 경우 (음변화 = 음편): 예를 들어 本은 보통 ほん이지만, 1본은 いっぽん이 됩니다.
  2. 어떤 사물에 어떤 카운터를 사용하는지 기억하기: 연필은 本? 個?

이 장에서는 가장 중요한 조수사를 하나씩 정복하겠습니다. 각 조수사마다 1부터 10까지의 완전한 활용 차트를 제공합니다.

6B: 고유 つ 체계 — 만능 안전망

일본어에는 고유 숫자 체계(和語数詞)가 있으며, 여기에 つ를 붙이면 거의 모든 사물을 셀 수 있는 만능 카운터가 됩니다. 정확한 조수사를 모를 때 이것을 사용하면 대부분 통합니다.

주의

つ 체계는 1~10까지만 존재합니다. 11 이상은 한자어 숫자 + 적절한 조수사를 사용해야 합니다. 또한, 사람을 셀 때는 つ를 사용하지 않습니다.

つ 카운터 완전 차트 (1~10)

숫자일본어읽기한국어 뜻
1一つひとつ (hitotsu)하나
2二つふたつ (futatsu)둘
3三つみっつ (mittsu)셋
4四つよっつ (yottsu)넷
5五ついつつ (itsutsu)다섯
6六つむっつ (muttsu)여섯
7七つななつ (nanatsu)일곱
8八つやっつ (yattsu)여덟
9九つここのつ (kokonotsu)아홉
10十とお (too)열
?いくついくつ (ikutsu)몇 개? / 몇 살?

패턴 관찰

3, 6, 8에서 っ(촉음)이 나타납니다: みっつ, むっつ, やっつ. 그리고 10은 つ가 붙지 않고 とお만 씁니다. いくつ는 "몇 개?"라는 의문사로, 나이를 물을 때도 사용합니다 (おいくつですか = 몇 살이세요?).

사용 예:

  • りんごを三つください。 — 사과를 세 개 주세요.
  • みかんが五つあります。 — 귤이 다섯 개 있습니다.
  • お子さんはおいくつですか。 — 자녀분은 몇 살인가요?

6C: 사람 카운터 人 — 완전 차트 (1~10)

사람을 세는 카운터는 人입니다. 한국어의 명 또는 사람에 해당합니다. 1명과 2명은 고유 일본어 읽기를 사용하며, 3명부터는 한자어 숫자 + にん을 사용합니다.

人(にん) 카운터 완전 차트

숫자일본어읽기한국어비고
1一人ひとり (hitori)한 명특수 읽기!
2二人ふたり (futari)두 명특수 읽기!
3三人さんにん (sannin)세 명
4四人よにん (yonin)네 명よ 사용
5五人ごにん (gonin)다섯 명
6六人ろくにん (rokunin)여섯 명
7七人しちにん / ななにん일곱 명두 읽기 모두 가능
8八人はちにん (hachinin)여덟 명
9九人きゅうにん / くにん아홉 명두 읽기 모두 가능
10十人じゅうにん (juunin)열 명
?何人なんにん (nannin)몇 명?

필수 암기: 1명과 2명

一人(ひとり)과 二人(ふたり)는 절대 규칙적 패턴이 아닙니다. いちにん, ににん이라고 읽지 않습니다! 이 두 단어는 개별적으로 완전히 외워야 합니다. 매우 자주 사용되므로 반복 노출을 통해 자연스럽게 익혀지기도 합니다.

사용 예:

  • 一人で来ました。 — 혼자 왔습니다.
  • 家族は四人です。 — 가족은 네 명입니다.
  • 何人ですか。 — 몇 명입니까? (식당에서 자주 듣는 질문)

6D: 작은 물건 個(こ) — 완전 차트 (1~10)

작고 둥글거나 형태가 뚜렷한 물건을 셀 때 사용합니다: 사과, 계란, 돌, 상자 등. 한국어의 개에 해당하며, 한자 個는 한국 한자음으로 개입니다. 동족어

個(こ) 카운터 완전 차트

숫자일본어읽기한국어비고
1一個いっこ (ikko)한 개いち → いっ (촉음)
2二個にこ (niko)두 개
3三個さんこ (sanko)세 개
4四個よんこ (yonko)네 개
5五個ごこ (goko)다섯 개
6六個ろっこ (rokko)여섯 개ろく → ろっ (촉음)
7七個ななこ (nanako)일곱 개
8八個はっこ (hakko)여덟 개はち → はっ (촉음)
9九個きゅうこ (kyuuko)아홉 개
10十個じゅっこ / じっこ열 개じゅう → じゅっ/じっ (촉음)
?何個なんこ (nanko)몇 개?

음변화 패턴: k행 카운터

個(こ)는 k행으로 시작하는 카운터입니다. k행 카운터에서는 1, 6, 8, 10에서 촉음(っ)이 들어갑니다: いっこ, ろっこ, はっこ, じゅっこ. 이 패턴은 階(かい), 回(かい) 등 다른 k행 카운터에도 적용됩니다.

6E: 길고 가는 물건 本(ほん) — 완전 차트 (1~10)

길고 가늘거나 원통형인 물건을 셀 때 사용합니다: 연필, 우산, 나무, 병, 도로, 강, 넥타이, 전화 통화 등. 한국어에서 가장 가까운 대응은 맥락에 따라 자루(연필), 병(음료), 그루(나무) 등 다양합니다.

本은 h행 카운터의 대표 주자이며, 가장 복잡한 음변화 패턴을 보입니다.

本(ほん) 카운터 완전 차트

숫자일본어읽기한국어음변화
1一本いっぽん (ippon)한 자루/병h → p (촉음 + 반탁음)
2二本にほん (nihon)두 자루/병변화 없음
3三本さんぼん (sanbon)세 자루/병h → b (탁음)
4四本よんほん (yonhon)네 자루/병변화 없음
5五本ごほん (gohon)다섯 자루/병변화 없음
6六本ろっぽん (roppon)여섯 자루/병h → p (촉음 + 반탁음)
7七本ななほん (nanahon)일곱 자루/병변화 없음
8八本はっぽん (happon)여덟 자루/병h → p (촉음 + 반탁음)
9九本きゅうほん (kyuuhon)아홉 자루/병변화 없음
10十本じゅっぽん (juppon)열 자루/병h → p (촉음 + 반탁음)
?何本なんぼん (nanbon)몇 자루/병?h → b (탁음)

음변화 기억법: "1, 6, 8, 10은 빵빵 (p), 3과 ?는 부드러운 b"

빵빵 그룹 (h → p): 1(いっぽん), 6(ろっぽん), 8(はっぽん), 10(じゅっぽん) — 숫자 끝이 촉음(っ)으로 바뀌면서 ほ가 ぽ으로 바뀝니다.

부드러운 그룹 (h → b): 3(さんぼん), ?(なんぼん) — ん 뒤에서 ほ가 ぼ으로 바뀝니다.

나머지(2, 4, 5, 7, 9)는 그대로 ほん입니다.

사용 예:

  • ペンを三本買いました。 — 펜을 세 자루 샀습니다.
  • ビールを一本ください。 — 맥주 한 병 주세요.
  • 傘は何本ありますか。 — 우산은 몇 자루 있습니까?

6F: 평평한 것 枚(まい) — 완전 차트 (1~10)

평평하고 얇은 것을 셀 때 사용합니다: 종이, 셔츠, 접시, 피자 한 판, CD, 사진 등. 한국어의 장에 해당합니다.

좋은 소식: 枚는 음변화가 전혀 없습니다! 모든 숫자에서 まい 그대로 유지됩니다.

2二枚にまい (nimai)두 장
3三枚さんまい (sanmai)세 장
4四枚よんまい (yonmai)네 장
5五枚ごまい (gomai)다섯 장
6六枚ろくまい (rokumai)여섯 장
7七枚ななまい (nanamai)일곱 장
8八枚はちまい (hachimai)여덟 장
9九枚きゅうまい (kyuumai)아홉 장
10十枚じゅうまい (juumai)열 장
?何枚なんまい (nanmai)몇 장?

사용 예:

  • 切符を二枚ください。 — 표 두 장 주세요.
  • 写真を五枚撮りました。 — 사진을 다섯 장 찍었습니다.
  • Tシャツが何枚ありますか。 — 티셔츠가 몇 장 있습니까?

6G: 동물 카운터 — 작은 동물 匹(ひき) + 큰 동물 頭(とう)

한국어에서는 동물을 모두 마리로 세지만, 일본어에서는 크기에 따라 카운터가 달라집니다.

  • 匹(ひき) — 작은/중간 크기 동물: 고양이, 개, 물고기, 벌레, 개구리 등
  • 頭(とう) — 큰 동물: 소, 말, 코끼리, 고래 등 (사람이 안을 수 없을 정도로 큰 동물)

匹(ひき) 카운터 완전 차트 — 작은 동물

숫자일본어읽기한국어음변화
1一匹いっぴき (ippiki)한 마리h → p
2二匹にひき (nihiki)두 마리변화 없음
3三匹さんびき (sanbiki)세 마리h → b
4四匹よんひき (yonhiki)네 마리변화 없음
5五匹ごひき (gohiki)다섯 마리변화 없음
6六匹ろっぴき (roppiki)여섯 마리h → p
7七匹ななひき (nanahiki)일곱 마리변화 없음
8八匹はっぴき (happiki)여덟 마리h → p
9九匹きゅうひき (kyuuhiki)아홉 마리변화 없음
10十匹じゅっぴき (juppiki)열 마리h → p
?何匹なんびき (nanbiki)몇 마리?h → b

匹의 음변화는 本과 동일!

匹(ひき)도 h행 카운터이므로 本(ほん)과 완전히 같은 패턴이 적용됩니다. 1, 6, 8, 10 → p음화 / 3, ? → b음화. 이것이 청킹의 위력입니다!

숫자일본어읽기한국어비고
1一頭いっとう (ittou)한 마리 (대형)촉음
2二頭にとう (nitou)두 마리
3三頭さんとう (santou)세 마리
4四頭よんとう (yontou)네 마리
5五頭ごとう (gotou)다섯 마리
6六頭ろくとう (rokutou)여섯 마리
7七頭ななとう (nanatou)일곱 마리
8八頭はっとう (hattou)여덟 마리촉음
9九頭きゅうとう (kyuutou)아홉 마리
10十頭じゅっとう (juttou)열 마리촉음
?何頭なんとう (nantou)몇 마리?

사용 예:

  • 猫が三匹います。 — 고양이가 세 마리 있습니다.
  • 牛が二頭います。 — 소가 두 마리 있습니다.
  • 犬は何匹飼っていますか。 — 개를 몇 마리 키우고 있습니까?

6H: 음료와 컵 杯(はい) — 완전 차트 (1~10)

컵이나 그릇에 담긴 음료, 밥 등을 셀 때 사용합니다. 한국어의 잔(커피 한 잔), 그릇(밥 한 그릇), 공기(밥 한 공기)에 해당합니다. 한자 杯의 한국 한자음은 배입니다. 동족어

숫자일본어읽기한국어음변화

보너스: いっぱい의 이중 의미

一杯(いっぱい)는 "한 잔"이라는 뜻이지만, いっぱい라고 히라가나로 쓰면 "가득"이라는 의미도 됩니다. お腹いっぱい = 배부르다 (배가 가득 차다).

한국어와 비교: 자음 동화

한국어에도 비슷한 음운 현상이 있습니다. 일본어의 음변화가 낯설게 느껴질 때 이렇게 비교해 보세요.

  • 한국어: "한국말" → [한궁말] (비음화)
  • 한국어: "합니다" → [함니다] (비음화)
  • 한국어: "입학" → [이팍] (격음화)
  • 일본어: いち + ほん → いっぽん (촉음 + 반탁음)

발음의 편의를 위해 인접 자음이 서로 영향을 주는 현상은 두 언어 모두에 존재합니다. 원리는 같습니다 — 입이 편한 쪽으로 소리가 변하는 것입니다.

궁극의 요약: 음변화가 일어나는 숫자

1, 3, 6, 8, 10, ? — 이 여섯 자리만 주의하면 됩니다!

  • 1, 6, 8, 10: 촉음(っ)이 삽입되고, h행이면 p로 변함
  • 3, ?: ん 뒤에서 h → b 또는 k → g (일부 카운터)
  • 2, 4, 5, 7, 9: 거의 항상 변화 없음!

6N: 어순 — 조수사의 위치

일본어 문장에서 "숫자 + 조수사"는 두 가지 위치에 올 수 있습니다.

패턴 1: 명사 + 수량 (가장 자연스러운 어순)

명사를 먼저 말하고, 조수사가 동사 바로 앞에 옵니다.

  • りんごを三個買いました。
  • 사과를 세 개 샀습니다.

구조: [명사]を [숫자+조수사] [동사]

패턴 2: 숫자 + 조수사 + の + 명사

수량을 먼저 말하고 の로 명사와 연결합니다. 수량을 강조할 때 사용합니다.

  • 三個のりんごを買いました。
  • 세 개의 사과를 샀습니다.

구조: [숫자+조수사]の [명사]を [동사]

어떤 패턴을 사용할까?

패턴 1이 일상 회화에서 훨씬 자연스럽고 자주 사용됩니다. 패턴 2는 글이나 형식적 표현에서 더 많이 보입니다.

한국어와 비교하면:

  • 패턴 1 = "사과를 세 개 샀어" (자연스러운 어순)
  • 패턴 2 = "세 개의 사과를 샀어" (강조 어순)

한국어에서도 두 패턴 모두 가능하므로, 감각적으로 이해하기 쉬울 것입니다.

존재 표현에서의 조수사

います(있습니다 - 생물) / あります(있습니다 - 사물)와 함께 사용할 때:

  • 猫が三匹います。 — 고양이가 세 마리 있습니다.
  • 本が五冊あります。 — 책이 다섯 권 있습니다.
  • 学生が二十人います。 — 학생이 스무 명 있습니다.

동족어의 힘

위 16개의 조수사 중 8개가 한국어와 같은 한자에서 유래했습니다. 절반이 동족어라는 뜻입니다! 한자를 아는 한국어 화자에게 이것은 엄청난 이점입니다. 台가 대이고, 回가 회라는 것을 알면, 새로운 단어를 배우는 것이 아니라 이미 아는 단어의 일본어 발음만 익히는 것입니다.

6P: 실전 조수사 — 생존 상황

실제 일본에서 마주칠 수 있는 상황별 표현을 익혀 봅시다. 각 표현에서 조수사가 어떻게 사용되는지 주목하세요.

상황 1: 식당에서

일본어한국어
何人ですか。몇 명이세요? (입구에서 점원이 묻는 말)
二人です。두 명입니다.
ビールを三杯ください。맥주 세 잔 주세요.
おにぎりを二個お願いします。주먹밥 두 개 부탁합니다.

상황 2: 쇼핑할 때

일본어한국어
このTシャツを二枚ください。이 티셔츠 두 장 주세요.
靴を一足試してもいいですか。신발 한 켤레 신어 봐도 될까요?
りんごを五個ください。사과 다섯 개 주세요.

상황 3: 호텔에서

일본어한국어
一人で泊まります。혼자 묵겠습니다.
三泊お願いします。3박 부탁합니다.

상황 4: 교통 / 이동

일본어한국어
切符を二枚お願いします。표 두 장 부탁합니다.
次の電車は何分ですか。다음 전차는 몇 분인가요?

상황 5: 일상 대화

일본어한국어
兄弟は何人いますか。형제가 몇 명 있습니까?
猫を二匹飼っています。고양이를 두 마리 키우고 있습니다.
この本を三回読みました。이 책을 세 번 읽었습니다.
お子さんは何歳ですか。자녀분은 몇 살인가요?
五歳です。다섯 살입니다.

상황 6: 건물에서

일본어한국어
トイレは何階ですか。화장실은 몇 층인가요?
三階です。3층입니다.

퀴즈 A: 조수사 매칭

왼쪽의 사물을 오른쪽의 올바른 조수사에 연결하세요.

사물조수사
1. りんご (사과)A. 匹
2. 猫 (고양이)B. 冊
3. 本 (책)C. 個
4. ペン (펜)D. 枚
5. 紙 (종이)E. 本
6. コーヒー (커피)F. 台
7. 車 (자동차)G. 杯

정답 보기

1-C, 2-A, 3-B, 4-E, 5-D, 6-G, 7-F, 8-H

  • 고양이(작은 동물) → 匹

  • 펜(길고 가는 것) → 本
  • 종이(평평한 것) → 枚
  • 커피(음료) → 杯
  • 자동차(기계/차량) → 台
  • 신발(짝) → 足

퀴즈 D: 실전 상황 — 다지선다

각 상황에 가장 적절한 표현을 고르세요.

1. 식당에서 점원이 "몇 분이세요?"라고 묻습니다. 3명일 때 대답은?

  • a) 三つです。
  • b) 三人です。
  • c) 三匹です。
  • d) 三個です。

2. 편의점에서 주먹밥을 2개 사고 싶습니다. 뭐라고 말할까요?

  • a) おにぎりを二枚ください。
  • b) おにぎりを二本ください。
  • c) おにぎりを二個ください。
  • d) おにぎりを二冊ください。

3. "우산 3자루"를 올바르게 읽은 것은?

  • a) 傘三本（さんほん）
  • b) 傘三本（さんぼん）
  • c) 傘三本（さんぽん）
  • d) 傘三枚（さんまい）

4. "개를 한 마리 키우고 있어요"를 일본어로?

  • a) 犬を一匹（いちひき）飼っています。
  • b) 犬を一匹（いっぴき）飼っています。
  • c) 犬を一頭（いっとう）飼っています。
  • d) 犬を一個（いっこ）飼っています。

5. 엘리베이터에서 "몇 층이세요?"라고 물어볼 때?

  • a) 何回ですか。
  • b) 何個ですか。
  • c) 何階ですか。
  • d) 何本ですか。
정답 보기
  1. b) 三人（さんにん）です。 — 사람은 人으로 셉니다.
  2. c) おにぎりを二個（にこ）ください。 — 주먹밥은 작고 둥근 물건이므로 個.
  3. b) 傘三本（さんぼん） — 3은 h → b 음변화가 적용됩니다.
  4. b) 犬を一匹（いっぴき）飼っています。 — 개는 작은 동물이므로 匹, 1은 h → p 음변화.
  5. c) 何階（なんがい）ですか。 — 건물 층은 階.

퀴즈 E: 문장 만들기

주어진 단어를 올바른 순서로 배열하여 자연스러운 일본어 문장을 만드세요.

  2. 猫が二匹います。 — 고양이가 두 마리 있습니다.

  4. 何人ですか。 — 몇 명입니까?
  5. 本が何冊ありますか。 — 책이 몇 권 있습니까?
