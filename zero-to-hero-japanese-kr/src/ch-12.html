기억 기법 스포트라이트: 전이 활성화

한자어 동형어를 통해 기존 한국어 어휘에서 일본어 어휘로 다리를 놓습니다.

한국어와 일본어는 한자에서 유래한 어휘의 40~60%를 공유합니다. 이 장에서는 그 공유 어휘를 체계적으로 활용하여 일본어 어휘력을 폭발적으로 확장합니다. 한국어 한자어 지식은 여러분의 가장 강력한 무기입니다.

"한국어 한자어를 알면 일본어 어휘의 절반은 이미 여러분의 것입니다. 음운 대응 규칙을 배우면 나머지 절반도 예측할 수 있습니다."

1. 확장 음운 대응 규칙 — 한국어로부터 일본어 읽기 예측하기

제11장에서 기본적인 음운 대응을 배웠습니다. 이제 각 규칙을 더 많은 예시와 함께 확장합니다. 이 규칙들을 완벽히 익히면 한국어 한자 읽기로부터 일본어 음독을 예측할 수 있습니다.

정교한 질문법

왜 한국어 ㅂ이 일본어 は행(h)에 대응할까요?

역사적으로 일본어 は행은 p → ɸ → h로 음운 변화를 겪었습니다. 한자가 처음 전래되었을 때 일본어 は행은 아직 [p]로 발음되었기 때문에, 한국어 ㅂ [p/b]와 같은 음이 사용되었습니다. 시간이 지나면서 일본어 쪽만 h로 변했고, 한국어는 b/p 음을 유지했습니다.

한국어 초성 자음과 일본어 가나 행의 대응표. ㅎ과 ㄱ 모두 か행에 대응합니다. ㅈ은 さ행과 た행에 걸쳐 있습니다.

3. 거짓 친구 (偽りの友) — 같아 보이지만 뜻이 다른 단어들

한자 동계어에서 가장 위험한 함정은 같은 한자를 쓰면서도 한국어와 일본어에서 다른 의미를 가진 단어입니다. 아래 30개 이상의 "거짓 친구"를 반드시 암기하십시오.

경고: 거짓 친구는 의사소통 사고를 유발합니다

같은 한자가 같은 뜻이라고 가정하면 심각한 오해가 발생할 수 있습니다. 위험도가 높을수록 더 주의 깊게 학습해야 합니다.

漢字한국어 의미일본어 의미위험도일본어 예문
大丈夫 대장부 (훌륭한 사나이) だいじょうぶ (괜찮다, 문제없다) 위험 大丈夫ですか？ = 괜찮으세요?
勉強 면강 (억지로, 겨우) べんきょう (공부, 학습) 위험 毎日勉強します。 = 매일 공부합니다.
丈夫 장부 (장부, 회계 장부) じょうぶ (튼튼한, 견고한) 위험 この靴は丈夫だ。 = 이 신발은 튼튼하다.
手紙 수지 (화장지, 휴지) てがみ (편지) 위험 手紙を書く。 = 편지를 쓰다.
汽車 기차 (기차) きしゃ (증기 기관차) 주의 汽車に乗る。 = 증기 기관차를 타다.
人参 인삼 (고려 인삼) にんじん (당근) 위험 人参を切る。 = 당근을 자르다.
無理 무리 (무리한, 과로) むり (불가능한, 무리한) 주의 無理しないでね。 = 무리하지 마세요.
愛人 애인 (연인, 사랑하는 사람) あいじん (정부/불륜 상대) 위험 일본에서는 "연인"을 恋人(こいびと)라고 합니다
外出 외출 (외출) がいしゅつ (외출) 안전 外出する。 = 외출하다. (같은 의미)
新聞 신문 (새로운 소식) しんぶん (신문) 주의 新聞を読む。 = 신문을 읽다.
経理 경리 (회계) けいり (회계) 안전 経理の仕事。 = 회계 업무.

切手 절수 (자르다?) きって (우표) 위험 切手を貼る。 = 우표를 붙이다.
床 상 (침대, 받침대) ゆか (바닥) 위험 床に座る。 = 바닥에 앉다.
質問 질문 (질문) しつもん (질문) 안전 質問があります。 = 질문이 있습니다.

정교한 질문법

왜 같은 한자가 한국어와 일본어에서 다른 의미를 가질까요?

한자가 전래된 이후 각 나라에서 독자적인 의미 변화가 일어났기 때문입니다. 예를 들어, 勉強은 원래 중국어에서 "마지못해 하다"라는 뜻이었습니다. 일본에서는 "마지못해서라도 하는 것 = 공부"로 의미가 확장되었고, 한국어에서는 원래의 "겨우, 억지로"라는 뜻을 유지했습니다.

4. 합성어 만들기 — 한자 구성 요소로 의미 추론하기

개별 한자의 의미를 알면 낯선 합성어의 의미를 추론할 수 있습니다. 이것이 한자 학습의 진정한 위력입니다.

핵심 원리: 한자 = 레고 블록

대부분의 한자어는 2~3개 한자의 조합으로 만들어집니다. 각 한자의 의미를 알면 전체 의미를 조립할 수 있습니다.

구성 요소 분석 예시 1

図書館

図(도면) + 書(글) + 館(관)
= 도면과 글을 보관하는 관 = 도서관

구성 요소 분석 예시 2

自動販売機

自(스스로) + 動(움직이다) + 販(팔다) + 売(팔다) + 機(기계)
= 스스로 파는 기계 = 자동판매기

구성 요소 분석 예시 3

不動産

不(아닌) + 動(움직이다) + 産(재산)
= 움직이지 않는 재산 = 부동산

구성 요소 분석 예시 4

心理学

心(마음) + 理(이치/원리) + 学(학문)
= 마음의 원리를 연구하는 학문 = 심리학

실전 팁: 모르는 단어를 만났을 때

글을 읽다가 모르는 한자어를 만나면 각 한자를 한국어 한자 읽기로 읽어 보십시오. 한국어 단어가 떠오르면 의미를 즉시 알 수 있습니다. 예: 国際連合 → 한국어로 읽기: 국제연합 → 바로 유엔입니다!

한자 구성 요소를 알면 한국어 지식으로 의미를 추론하고 음운 대응 규칙으로 일본어 읽기를 예측할 수 있습니다.
