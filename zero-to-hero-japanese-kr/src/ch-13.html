제13장경어와 공손함 — 존댓말에서 敬語로

제5장에서 ください와 お願いします를 경어인 줄 모르고 사용했습니다. 제8장에서는 자기소개에서 申します를 사용했습니다. 이미 경어를 사용해 왔으니, 이제 전체 체계를 이해해 봅시다.

"한국어 화자는 이미 경어 체계 안에서 살고 있습니다. 언제 말투를 바꿔야 하는지, 누가 윗사람인지, 하나의 어미가 어떻게 사회적 상황을 바꾸는지 이미 알고 있습니다. 일본어 경어(敬語)는 같은 영역의 다른 지도입니다."

복습 확인 — 제8장~제12장

경어 체계를 배우기 전에, 이전 장에서 배운 조사, 활용, 한자, 어휘가 아직 확실한지 확인합니다.

제1절: 구조적 대조 — 한국어의 경어 단계와 일본어 경어(敬語)

한국어에는 풍부한 경어 체계가 있습니다. 일본어에도 이에 대응하는 체계가 있지만, 범주의 대응 관계가 다릅니다. 핵심적인 대응 관계는 다음과 같습니다:

한국어

한국어 경어 체계

↔

일본어

日本語 敬語体系

한국어 단계일본어 대응 표현기능예시
반말 (비격식) タメ口 / 普通体 동등한 관계에서 사용하는 기본형/사전형 食べる、行く、いい
존댓말 (공손) 丁寧語 (ていねいご) 공손한 어미: -です / -ます 食べます、行きます、いいです
높임말 (존경) 尊敬語 (そんけいご) 상대방의 행위를 높임 お+になる、召し上がる
겸양어 (겸손) 謙譲語 (けんじょうご) 자신의 행위를 낮춤 お+する、申す、いただく
한국어 화자의 이점

여러분은 이미 그 논리를 이해하고 있습니다: 상대를 높이고, 자신을 낮추며, 상황에 따라 어미를 조절합니다. 한국어에서 해체, 해요체, 하십시오체를 자동으로 전환하듯, 일본어에서는 普通体, 丁寧語, 尊敬語/謙譲語를 전환합니다. 감각은 같고, 형태만 다릅니다.

4단계 경어 대조도. 한국어와 일본어는 같은 개념적 구조를 공유합니다 — 비격식, 공손, 존경(상대를 높임), 겸양(자신을 낮춤).
정교화 질문

왜 한국어는 높임말을 어미에 삽입(-시-)하는 반면, 일본어는 완전히 다른 동사(召し上がる vs 食べる)를 사용할까요?

한국어는 높임을 교착적으로 삽입합니다: 먹다 → 드시다, 가다 → 가시다 — 시 삽입어는 생산적이고 예측 가능합니다. 일본어도 お～になる 패턴으로 이렇게 하지만, 기본형과 형태론적 관계가 전혀 없는 불규칙 "특수 경어 동사" 목록도 유지합니다. 이 부분은 한국어의 직감으로 해결되지 않으므로, 쌍을 단순히 암기해야 합니다.

제2절: 특수 경어 동사 쌍과 한국어 대응 표현

이 동사들은 예측 가능한 패턴을 따르지 않는 불규칙 경어 동사입니다. 어릴 때 먹다 → 드시다를 외웠듯이, 고정된 쌍으로 암기해야 합니다. 아래 표는 기본형 → 존경어(尊敬語) → 겸양어(謙譲語) → 한국어 대응을 보여줍니다.

기본형尊敬語 (존경)謙譲語 (겸양)한국어 기본형한국어 높임말한국어 겸양어
食べる 召し上がる いただく 먹다 드시다 먹겠습니다
飲む 召し上がる いただく 마시다 드시다 마시겠습니다
行く いらっしゃる 参る (まいる) 가다 가시다 가겠습니다
来る いらっしゃる 参る (まいる) 오다 오시다 오겠습니다
いる いらっしゃる おる 있다 계시다 있겠습니다
言う おっしゃる 申す (もうす) 말하다 말씀하시다 말씀드리다
する なさる いたす 하다 하시다 하겠습니다
見る ご覧になる (ごらんになる) 拝見する (はいけんする) 보다 보시다 봬다 / 뵙다
知る ご存じだ (ごぞんじだ) 存じる (ぞんじる) 알다 아시다 알고 있습니다
もらう — いただく 받다 — 받겠습니다
あげる — 差し上げる (さしあげる) 주다 — 드리다
패턴을 주목하세요

いらっしゃる는 존경어의 만능 도구입니다 — 行く, 来る, いる를 모두 포괄합니다. 마찬가지로 いただく는 겸양형에서 食べる와 もらう를 모두 포괄합니다. 문맥이 의미를 결정하며, 이는 한국어에서 계시다가 "계시다"와 "머무르시다"를 모두 포괄하는 것과 같습니다.

암기법: 핵심 5쌍

이 5개의 존경어/겸양어 쌍을 먼저 우선적으로 외우세요. 거의 모든 비즈니스 또는 격식 있는 상황에서 등장합니다:
1. いらっしゃる / 参る — 있다, 가다, 오다
2. おっしゃる / 申す — 말하다
3. 召し上がる / いただく — 먹다, 받다
4. ご覧になる / 拝見する — 보다
5. なさる / いたす — 하다

존경어는 상대방의 행위를 높이고, 겸양어는 화자의 행위를 낮춥니다. 효과는 동일합니다: 존경의 거리를 넓히는 것입니다. 한국어도 같은 이축 체계를 사용합니다.

제3절: お/ご 접두사 체계

일본어에는 명사와 형용사 앞에 お 또는 ご를 붙여 더 공손하게 만드는 체계적인 방법이 있습니다. 규칙은 단어의 어원에 기반합니다:

お + 和語 (일본 고유어)

고대 일본어에서 유래한 단어(和語/やまとことば)에는 お를 붙입니다:

はし → おはし젓가락
水 → お水물
名前 → お名前이름
茶 → お茶차
金 → お金돈
手紙 → お手紙편지
体 → お体몸/건강

ご + 漢語 (한자어)

중국에서 차용한 단어(漢語/かんご)에는 ご를 붙입니다:

住所 → ご住所주소
連絡 → ご連絡연락
家族 → ご家族가족
意見 → ご意見의견
案内 → ご案内안내
質問 → ご質問질문
注文 → ご注文주문
한국어 대응: 고유 경어 명사

한국어에는 お/ご와 같은 접두사 체계가 없습니다. 대신 한국어에는 완전히 별개의 경어 명사가 있습니다:
밥 → 진지 (식사), 말 → 말씀 (말), 나이 → 연세 (나이), 이름 → 성함 (이름), 집 → 댁 (집).
일본어는 더 체계적입니다: お/ご를 붙이면 단어가 격식을 갖춥니다. 한국어는 단어 자체를 교체합니다.

お/ご + 동사 패턴 (생산적 경어)

특수 동사 쌍을 암기하는 것 외에도, 이 공식을 사용하면 어떤 동사든 경어를 만들 수 있습니다:

유형공식예시 (待つ = 기다리다)한국어 대응
尊敬語 お + 連用形 + になる お待ちになる 기다리시다
謙譲語 お + 連用形 + する お待ちする 기다리겠습니다

생산적 패턴의 추가 예시

기본형尊敬語 (お～になる)謙譲語 (お～する)
読むお読みになるお読みする
書くお書きになるお書きする
聞くお聞きになるお聞きする
話すお話しになるお話しする
帰るお帰りになる—
伝えるお伝えになるお伝えする

이 생산적 패턴은 한국어에서 -시-를 붙이는 것과 같습니다 — 거의 모든 동사에 적용됩니다. 제2절의 불규칙 특수 쌍은 이 패턴보다 우선합니다.

정교화 질문

왜 일본어는 경어 접두사를 어원별로(お는 和語, ご는 漢語) 구분하는데 한국어에는 그런 구분이 없을까요?

일본어는 문법과 음운 체계 전반에 걸쳐 고유어(和語)와 한자 차용어(漢語)를 깊이 구분합니다. お/ご 구분은 이 어원적 인식의 표면적 반영입니다. 한국어는 한자어를 너무 완전히 흡수하여 고유어/차용어 구분이 같은 방식으로 문법 규칙을 좌우하지 않습니다. 이것은 한국어가 매끄럽게 다듬은 역사적 층위를 일본어가 보존하고 있는 또 다른 사례입니다.

제4절: 상황별 연습

경어는 맥락 속에서만 의미가 있습니다. 어떤 단계를 사용해야 하는지, 그리고 그 이유를 보여주는 네 가지 실제 상황을 소개합니다.

상황 1: 회사에서 — 상사의 상사를 만날 때

여러분은 신입사원(新入社員)입니다. 부장(部長)을 처음 만납니다.

상황일본어 (경어)단계한국어 대응
자기소개 初めまして。田中と申します。よろしくお願いいたします。 謙譲語 처음 뵙겠습니다. 다나카라고 합니다. 잘 부탁드리겠습니다.
보고서를 보셨는지 묻기 報告書はご覧になりましたか。 尊敬語 보고서는 보셨습니까?
확인하겠다고 말하기 私が確認いたします。 謙譲語 제가 확인하겠습니다.
어떻게 생각하시는지 묻기 部長はどのようにお考えですか。 尊敬語 부장님은 어떻게 생각하십니까?

상황 2: 레스토랑에서 — 직원이 손님에게 말할 때

여러분은 서버입니다. 손님은 항상 높여야 합니다.

상황일본어 (경어)단계한국어 대응
손님 맞이 いらっしゃいませ。何名様でいらっしゃいますか。 尊敬語 어서 오십시오. 몇 분이십니까?
주문 받기 ご注文はお決まりでしょうか。 尊敬語 주문하시겠습니까?
음식 서빙 お待たせいたしました。どうぞ召し上がってください。 謙+尊 오래 기다리셨습니다. 드시기 바랍니다.
계산서 가져오기 お会計はこちらでございます。 丁寧+謙 계산은 이쪽입니다.

상황 3: 교수님에게 보내는 이메일

대학교 교수님에게 이메일을 씁니다. 서면 경어는 구어보다 더 격식적입니다.

件名：来週のゼミについてお伺いしたいことがございます

山田先生

お忙しいところ失礼いたします。
来週のゼミの発表についてお伺いしたいことがございます。
お時間のある時に、ご連絡いただけますと幸いです。

何卒よろしくお願い申し上げます。

田中太郎

한국어 대응 격식: 교수님, 바쁘신 중에 죄송합니다. 다음 주 세미나 발표에 대해 여쭈어 볼 것이 있습니다. 시간 되실 때 연락 주시면 감사하겠습니다. 잘 부탁드립니다.

상황 4: 고객센터에 전화할 때

회사에 전화합니다. 상담원은 고객인 여러분에게 최고 수준의 경어를 사용합니다.

상담원의 말단계의미 (기본형)
お電話ありがとうございます。 丁寧語 電話ありがとう。
少々お待ちくださいませ。 尊敬語 ちょっと待って。
お名前をお伺いしてもよろしいでしょうか。 謙譲語 名前を聞いてもいい？
担当者にお繋ぎいたします。 謙譲語 担当者に繋ぐ。
ご不明な点がございましたら、お申し付けくださいませ。 謙+尊 分からないことがあったら、言ってね。

제5절: 외국인이 흔히 저지르는 경어 실수

상급 학습자도 이런 실수를 합니다. 한국어 배경은 일부 실수를 방지해 주지만, 전부는 아닙니다.

실수 이중 경어 (二重敬語)

✗ お召し上がりになる

✓ 召し上がる

召し上がる는 이미 존경어입니다. 여기에 お～になる를 더하면 이중 경어가 됩니다. 한국어에서 *드시시다라고 절대 하지 않는 것과 같습니다.

실수 자기 행위에 존경어 사용

✗ 私がいらっしゃいます

✓ 私が参ります

いらっしゃる는 주어를 높입니다. 자신에게 사용하면 한국어에서 *제가 가시겠습니다라고 하는 것처럼 문법적으로 말이 되지 않습니다.

실수 자사를 높이는 표현

✗ 私の会社の社長がおっしゃいました

✓ 弊社の社長が申しました

외부인에게 이야기할 때, 상사를 포함한 회사 내 모든 사람은 "우리 편"이므로 겸양어를 사용합니다. 한국어에서도 마찬가지입니다: 저희 사장님이 말씀하셨습니다가 격식적인 외부 맥락에서는 저희 사장이 말씀드렸습니다가 됩니다.

실수 ございます를 아무 데나 사용

✗ それはすごくございます

✓ それは素晴らしゅうございます (또는 간단히 素晴らしいです)

ございます는 ある의 경어형이지, 범용 공손함 강화제가 아닙니다. 무엇에 붙일 수 있는지에 대한 엄격한 문법 규칙이 있습니다.

실수 문장 중간에 격식 수준 혼합

✗ 先生が来たんですが、もう帰りになりました

✓ 先生がいらっしゃったんですが、もうお帰りになりました

문장 전체에서 격식 수준을 일관되게 유지하세요. 같은 사람에 대해 한 동사에서 존경어를 사용했다면, 모든 동사에서 사용해야 합니다.

실수 비격식 상황에서 과도한 경어

친한 친구나 어린 가족에게 경어를 사용하면 딱딱하고 거리감이 생깁니다. 한국어에서 가장 친한 친구에게 하십시오체로 말하면 이상하게 느끼듯이, 친한 친구에게 召し上がりますか를 사용하는 것도 마찬가지로 부자연스럽습니다. 상황을 파악하세요.
