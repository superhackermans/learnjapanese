1. 용어집 — Glossary

문자 체계 (Scripts)

ひらがな
히라가나. 46 기본 문자. 일본어 고유어, 활용 어미, 조사 표기에 사용.
カタカナ
가타카나. 46 기본 문자. 외래어, 의성어/의태어, 강조 표기에 사용.
漢字
한자. 중국에서 유래한 표의문자. 상용한자 2,136자가 기본.
音読み
おんよみ. 한자의 중국어 유래 읽기. 한국어 한자음과 대응.
訓読み
くんよみ. 한자의 일본어 고유 읽기. 한국어와 대응하지 않음.
送り仮名
おくりがな. 한자 뒤에 붙는 히라가나. 활용형을 표시.
振り仮名 / ルビ
ふりがな. 한자 위/옆에 표기하는 읽기 안내.
常用漢字
じょうようかんじ. 일본 정부 지정 일반 사용 한자 2,136자.

문법 용어 (Grammar Terms)

助詞
じょし. 조사. 한국어의 조사와 동일한 기능. は, が, を, に, で, へ, の 등.
動詞
どうし. 동사. 한국어와 동일하게 문장 끝에 위치 (SOV).
形容詞
けいようし. 형용사. い-형용사와 な-형용사 두 종류.
名詞
めいし. 명사. 한국어와 달리 격변화 없이 조사로 문법 기능 표시.
副詞
ふくし. 부사. とても, すぐに, たぶん 등.
接続詞
せつぞくし. 접속사. しかし, だから, そして 등.
活用
かつよう. 활용/변형. 한국어의 어미 변화와 같은 개념.
五段動詞
ごだんどうし. 5단 동사 (1그룹). 어미가 う단으로 끝남. 書く, 読む, 話す 등.
一段動詞
いちだんどうし. 1단 동사 (2그룹). 어미가 る로 끝남 (え단/い단+る). 食べる, 見る 등.
敬語
けいご. 경어. 한국어 존댓말/반말 체계와 유사한 개념.
尊敬語
そんけいご. 존경어. 상대의 동작을 높이는 표현. 한국어의 -(으)시-에 대응.
謙譲語
けんじょうご. 겸양어. 자신의 동작을 낮추는 표현.
丁寧語
ていねいご. 정중어. です/ます 등 정중한 문체.
受身
うけみ. 수동태. -(ら)れる. 한국어 피동과 대응.
使役
しえき. 사역. -(さ)せる. 한국어 사동과 대응.
仮定形
かていけい. 가정형. ～ば, ～たら, ～なら 등.
連体形
れんたいけい. 연체형. 명사를 수식하는 동사/형용사 형태.
連用形
れんようけい. 연용형. 다른 요소에 연결되는 형태 (ます형의 기본).
テ形
てけい. て-형. 연결, 요청, 진행 등 다양한 기능. 한국어의 -아/어서와 유사.

어휘 분류 (Vocabulary Domains)

漢語
かんご. 한자어. 중국어에서 유래한 단어. 한국어 한자어와 대응.
和語
わご. 화어/야마토 말. 일본어 고유 단어. 한국어와 대응 없음.
外来語
がいらいご. 외래어. 주로 영어 등 서양어에서 차용. 가타카나로 표기.
混種語
こんしゅご. 혼종어. 한자어+고유어 등 여러 기원이 혼합된 단어.
擬音語
ぎおんご. 의성어. ワンワン(멍멍), ドキドキ(두근두근) 등.
擬態語
ぎたいご. 의태어. キラキラ(반짝반짝), ぐるぐる(빙글빙글) 등.
同形異義語
どうけいいぎご. 동형이의어 (False Friends). 같은 한자인데 의미가 다른 단어.

문화 개념 (Cultural Concepts)

建前
たてまえ. 공적 입장/표면적 의견. 한국어의 공식적 입장과 유사.
本音
ほんね. 진심/실제 생각. 한국어의 속마음과 대응.
空気を読む
くうきをよむ. 분위기를 읽다. 한국어 "눈치"와 유사한 개념.
遠慮
えんりょ. 사양/자제. 한국어 "사양하다"와 유사하나 일본에서 더 빈번히 사용.
お辞儀
おじぎ. 인사하는 절. 각도에 따라 의미 변화 (15도/30도/45도).
先輩・後輩
せんぱい/こうはい. 선배/후배. 한국어와 거의 동일한 개념.
年功序列
ねんこうじょれつ. 연공서열. 한국어 "연공서열"과 동일 개념+한자.

JLPT 레벨 (JLPT Levels)

N5
기초. ひらがな/カタカナ + 한자 ~100자. 기본 인사, 자기소개 수준.
N4
초급. 한자 ~300자, 어휘 ~1,500. 일상 기본 회화 수준.
N3
중급. 한자 ~650자, 어휘 ~3,500. 일상 회화 + 간단한 독해 가능.
N2
중상급. 한자 ~1,000자, 어휘 ~6,000. 취업/유학 기준. 뉴스 이해 가능.
N1
상급. 한자 ~2,000자, 어휘 ~10,000. 학술/전문 분야 활동 가능.

3. 음운 대응 치트시트 — Sound Correspondence Cheat Sheet

한국어 한자음과 일본어 음독 사이에는 체계적인 음운 대응 규칙이 있습니다. 이 규칙을 알면 모르는 일본어 한자어의 발음을 예측할 수 있습니다. (제10-11장 상세 참조)

사용법

한국어 한자음의 초성/중성/종성을 확인하고 아래 규칙을 적용하면 일본어 음독을 추측할 수 있습니다. 정확도 약 70-80%.

초성 대응 (Initial Consonants)

ㅎ → か행 (k)

한국어 ㅎ은 일본어 か행으로 대응됩니다.

학(學) → がく 한(韓) → かん 해(海) → かい 현(現) → げん 화(火) → か 확(確) → かく 환(還) → かん
ㄱ → か행 (k/g)

한국어 ㄱ은 일본어 か행으로 대응됩니다.

경(經) → けい 교(敎) → きょう 국(國) → こく 기(機) → き 구(區) → く 금(金) → きん 감(感) → かん
ㅈ → さ행 (s) / た행 (t)

한국어 ㅈ은 일본어에서 さ행 또는 た행으로 갈립니다.

자(自) → じ 전(全) → ぜん 정(政) → せい 제(制) → せい 중(中) → ちゅう 진(眞) → しん 지(地) → ち
ㅂ → は행 (h) / ば행 (b)

한국어 ㅂ은 일본어 は행 또는 ば행으로 대응됩니다.

반(半) → はん 발(發) → はつ 방(方) → ほう 병(病) → びょう 복(福) → ふく 분(分) → ぶん 비(比) → ひ
ㅅ → さ행 (s)

한국어 ㅅ은 일본어 さ행으로 대응됩니다.

산(山) → さん 생(生) → せい 선(先) → せん 성(成) → せい 시(市) → し 신(新) → しん 수(水) → すい
ㄷ → た행 (t/d)

한국어 ㄷ은 일본어 た행으로 대응됩니다.

대(大) → だい/たい 도(都) → と 독(獨) → どく 동(東) → とう 단(單) → たん 답(答) → とう
ㅁ → ま행 (m)

한국어 ㅁ은 일본어 ま행으로 대응됩니다.

문(文) → ぶん/もん 민(民) → みん 명(明) → めい 목(目) → もく 무(無) → む 만(萬) → まん
ㄴ → な행 (n)

한국어 ㄴ은 일본어 な행으로 대응됩니다.

남(南) → なん 년(年) → ねん 노(努) → ど 내(內) → ない 능(能) → のう 녀(女) → じょ
ㄹ → ら행 (r)

한국어 ㄹ은 일본어 ら행으로 대응됩니다.

리(理) → り 력(力) → りょく 료(料) → りょう 류(流) → りゅう 론(論) → ろん 랑(浪) → ろう

종성 대응 (Final Consonants / Coda)

ㄱ 받침 → く / き
학(學) → がく 국(國) → こく 독(獨) → どく 적(的) → てき 역(驛) → えき 식(食) → しょく
ㄹ 받침 → つ
발(發) → はつ 물(物) → ぶつ 일(日) → にち/じつ 출(出) → しゅつ 월(月) → げつ/がつ 실(室) → しつ
ㅂ 받침 → う (장모음) / ふ
업(業) → ぎょう 답(答) → とう 법(法) → ほう 입(入) → にゅう 십(十) → じゅう 급(級) → きゅう
ㄴ 받침 → ん
산(山) → さん 문(文) → ぶん 인(人) → じん/にん 전(全) → ぜん 안(安) → あん 원(院) → いん
ㅇ 받침 → う (장모음) / い
생(生) → せい 경(經) → けい 동(東) → とう 정(政) → せい 공(工) → こう 명(明) → めい

모음 대응 (Vowel Correspondences)

ㅏ → あ단
산 → さん 간 → かん 반 → はん 만 → まん 남 → なん
ㅗ → お단 / おう
도(都) → と 고(高) → こう 소(小) → しょう 보(報) → ほう 노(努) → ど
ㅜ → う단
무(無) → む 구(區) → く 부(部) → ぶ 수(水) → すい 주(主) → しゅ
ㅐ / ㅔ → あい / えい
대(大) → だい/たい 해(海) → かい 재(財) → ざい 제(制) → せい 세(世) → せい

4. 동형이의어 — False Friends (50+)

같은 한자를 사용하지만 의미가 다른 단어들입니다. 한국어 화자가 가장 많이 틀리는 함정이므로 반드시 확인하세요.

위험도 기준

위험 = 의미가 완전히 다름. 오해 가능성 높음.  |  주의 = 의미가 일부 다르거나 용법이 다름.  |  참고 = 미묘한 뉘앙스 차이.

한자한국어 의미일본어 의미위험도예문 / 설명
勉強면강하다 (억지로 하다)공부하다위험毎日勉強している = 매일 공부하고 있다 (억지가 아님)
丈夫장부 (남편/사나이)튼튼하다, 괜찮다위험大丈夫ですか = 괜찮으세요? (남편이 아님)
新聞새로운 소식신문 (newspaper)주의新聞を読む = 신문을 읽다 (한국어에서도 신문이지만 원뜻 차이)
手紙화장지/휴지편지위험手紙を書く = 편지를 쓰다 (화장지가 아님!)
汽車기차 (train)증기 자동차 / (옛) 자동차위험일본어에서 기차는 電車 또는 列車
外出외출외출참고의미는 같으나 일본어에서 がいしゅつ과 そとで 구분에 주의
娘낭자 (젊은 여인)딸위험娘がいます = 딸이 있습니다 (젊은 여인이 아님)
愛人애인 (연인)정부/내연관계위험일본어에서 연인/애인은 恋人(こいびと). 愛人은 부정적 의미
人参인삼당근위험人参を切る = 당근을 자르다. 인삼은 高麗人参
工夫공부하다궁리하다, 연구하다주의工夫する = 방법을 연구하다/궁리하다 (공부가 아님)
得意뜻을 이루다잘하는 것, 장기주의料理が得意です = 요리를 잘합니다
苦手(없음)서투른 것, 어려워하는 것주의数学が苦手です = 수학을 어려워합니다
返事반사 (돌려보냄)대답, 응답위험返事をする = 대답하다
迷惑미혹 (헤매어 혹하다)폐/민폐주의ご迷惑をおかけして = 폐를 끼쳐서
結構결구 (결국, 구조)꽤, 충분히, 괜찮습니다주의結構です = 괜찮습니다 / 됐습니다
我慢(없음)참다, 인내하다주의我慢できない = 참을 수 없다
邪魔사마 (삿된 마귀)방해주의お邪魔します = 실례합니다 (방문 시)
約束약속약속참고의미 동일하나 일본어에서 더 가벼운 약속에도 사용
出世출세 (세상에 나옴)출세 / 승진참고한국어와 비슷하나 일본어에서 직장 승진 의미가 더 강함
気分기분기분참고의미 동일. 気分が悪い = 기분이 나쁘다 / 속이 메스껍다 (이중 의미)
無理무리무리 / 불가능참고無理しないで = 무리하지 마세요
八方美人팔방미인 (다재다능)모든 사람에게 좋은 얼굴 (부정적)위험한국어에서는 칭찬, 일본어에서는 비판적 표현
高慢고만 (그만두다?)거만하다, 오만하다주의高慢な態度 = 거만한 태도
親切친절친절참고의미 동일. 용법도 유사
石鹸(없음)비누주의한자로 유추하기 어려운 단어. 암기 필요
留守유수 (남아 지키다)부재 / 외출 중위험留守にする = 집을 비우다 (지키는 것이 아님!)
先生선생선생 / 의사 / 변호사 등참고일본어에서 더 넓은 범위로 사용 (의사, 작가 등에도)
主人주인남편 / 주인주의ご主人 = (상대의) 남편. 한국어보다 "남편" 의미가 강함
家内가내 (집안)아내 (겸양)위험家内がいます = 아내가 있습니다
心配심배 (마음 나눔)걱정주의心配しないで = 걱정하지 마세요
面倒(없음)귀찮다, 번거롭다주의面倒くさい = 귀찮다
世話(없음)돌보다, 보살피다주의お世話になります = 신세를 지겠습니다
怪我(없음)부상, 다침위험怪我をした = 다쳤다 (나의 잘못이 아님)
真面目(없음)성실하다, 진지하다주의真面目な人 = 성실한 사람
大家대가 (거장)집주인위험大家さん = 집주인
切手(없음)우표위험切手を貼る = 우표를 붙이다 ("손 자르다"가 아님)
用心용심 (마음 씀)조심하다, 주의하다참고火の用心 = 불조심
皮肉피육 (가죽과 살)아이러니, 비꼬기위험皮肉を言う = 비꼬는 말을 하다
床상 (평상)바닥, 마루주의床に座る = 바닥에 앉다. 침대는 ベッド
朝飯前아침밥 전아주 쉬운 일주의そんなこと朝飯前だ = 그런 건 식은 죽 먹기다
適当적당적당 / 대충주의適当にやる = 대충 하다 (부정적 뉘앙스 가능)
遠慮원려 (먼 걱정)사양하다, 삼가다주의ご遠慮ください = 삼가 주십시오
正直정직정직, 솔직참고正直に言うと = 솔직히 말하면
交番교번 (번갈아 교대)파출소위험交番はどこですか = 파출소는 어디입니까?

油断유단 (기름 끊다?)방심하다주의油断しないで = 방심하지 마세요
