제3장가타카나 — 외래어 문자 (カタカナ)

기억 기법 스포트라이트: 이중 부호화

히라가나와 가타카나를 시각적으로 짝지어 두 가지 검색 경로를 만듭니다.

가타카나는 히라가나의 각진 쌍둥이입니다. 같은 46개 소리를 나타내지만, 생김새는 완전히 다릅니다. 히라가나가 둥글고 유려한 반면, 가타카나는 날카롭고 직선적입니다 -- 초서체가 아닌 한자의 일부에서 파생되었기 때문입니다. 외래어, 강조, 의성어/의태어, 과학 용어 등에 사용됩니다. 목표: 3~5일 (이미 소리를 알고 있으므로 히라가나보다 빠릅니다).

한국어의 이점: 콩글리시 = 지름길

한국어에는 영어 외래어(콩글리시)가 가득합니다: 커피, 텔레비전, 버스, 호텔... 일본어에도 가타카나로 쓰인 똑같은 단어가 있습니다! 가타카나를 읽을 수 있으면, 한국어에서 이미 알고 있는 수백 개의 일본어 단어를 즉시 인식할 수 있습니다. 이 장에서 그 능력을 드립니다.

가타카나는 언제 사용하나요?

1. 외래어: コーヒー (커피), テレビ (TV)
2. 외국 이름: ソウル (서울), キム (김)
3. 강조: 영어의 대문자나 굵은 글씨처럼 -- ダメ! (안 돼!)
4. 의성어/의태어: ドキドキ (두근두근), ワクワク (설렘)
5. 과학/기술 용어: ウイルス (바이러스), エネルギー (에너지)

5개의 모음 -- ア イ ウ エ オ

히라가나 あ い う え お와 같은 소리이지만, 더 날카로운 모양입니다. 쌍으로 함께 익히세요.

a -- 히라가나 짝: あ
한국어 근사값: 아
기억법: 옥상 위의 안테나 -- 각진 획이 TV 안테나처럼 보입니다. 둥근 あ와 비교해 보세요. 외래어: アイス (아이스) = 아이스크림 / 아이스.
ア
i -- 히라가나 짝: い
한국어 근사값: 이
기억법: 기울어진 알파벳 "I" 같은 모양 -- 두 개의 직선 획이 이젤처럼 서로 기대고 있습니다. 외래어: インターネット (인타아넷토) = 인터넷 / 인터넷.
イ
u -- 히라가나 짝: う
한국어 근사값: 우 (비원순)
기억법: 위를 향한 왕관 모양 -- 세 획이 왕관 형태를 이룹니다. 외래어: ウイスキー (우이스키이) = 위스키 / 위스키.
ウ
e -- 히라가나 짝: え
한국어 근사값: 에
기억법: 엘리베이터 -- 위아래의 수평선을 수직 축이 연결합니다. 대문자 "I"나 I형 빔처럼 보입니다. 외래어: エレベーター (에레베에타아) = 엘리베이터 / 엘리베이터.
エ
o -- 히라가나 짝: お
한국어 근사값: 오
기억법: 팔을 벌린 오페라 가수 -- 가로 획과 세로 획이 포즈를 취하는 사람처럼 보입니다. 외래어: オレンジ (오렌지) = 오렌지 / 오렌지.
オ

카행 -- カ キ ク ケ コ

"k" 자음 행입니다. 각각 히라가나 짝이 있습니다. 가타카나 버전이 더 각집니다.

カ
ka -- 히라가나 짝: か
한국어: 가/카
기억법: 날카로운 카타나(칼) -- 각진 획이 대각선으로 베어냅니다. 둥근 か와 비교해 보세요. 외래어: カメラ (카메라) = 카메라 / 카메라.
キ
ki -- 히라가나 짝: き
한국어: 기/키
기억법: 날카로운 이가 있는 키(열쇠) -- き에서 아래쪽 곡선이 없는 형태입니다. 외래어: キムチ (키무치) = 김치 / 김치.
ク
ku -- 히라가나 짝: く
한국어: 구/쿠
기억법: 각진 갈고리 모양 -- く가 날카로워진 것 같습니다. 외래어: クラス (쿠라스) = 클래스 / 클래스.
ケ
ke -- 히라가나 짝: け
한국어: 게/케
기억법: 주둥이가 있는 케틀(주전자) -- 가로 획이 손잡이, 대각선이 주둥이입니다. 외래어: ケーキ (케에키) = 케이크 / 케이크.
コ
ko -- 히라가나 짝: こ
한국어: 고/코
기억법: 코너(모퉁이) -- 두 개의 직각이 열린 상자를 만듭니다. こ를 세로로 쌓은 모양입니다. 외래어: コーヒー (코오히이) = 커피 / 커피.

사행 -- サ シ ス セ ソ

サ
sa -- 히라가나 짝: さ
한국어: 사
기억법: 사무라이의 교차된 칼 -- 날카로운 대각선 획입니다. 외래어: サラダ (사라다) = 샐러드 / 샐러드.
シ
shi -- 히라가나 짝: し
한국어: 시
기억법: 윙크하는 얼굴 -- 세 획이 옆으로 기울어진 웃는 얼굴을 만듭니다. 점이 눈이고 곡선이 미소입니다. 주의: ツ (츠)와 혼동하지 마세요! シ는 점이 아래에서 위로 가고 곡선이 더 수평적입니다. 외래어: シャツ (샤츠) = 셔츠 / 셔츠.
ス
su -- 히라가나 짝: す
한국어: 스
기억법: 스키 슬로프 -- 하나의 직선 획이 아래로 기울어집니다. 외래어: スプーン (스푸운) = 스푼 / 스푼.
セ
se -- 히라가나 짝: せ
한국어: 세
기억법: 올라가는 세단(계단) -- 각진 획이 위로 올라갑니다. 외래어: セーター (세에타아) = 스웨터 / 스웨터.
ソ
so -- 히라가나 짝: そ
한국어: 소
기억법: 소프트 아이스크림 -- 두 획이 아래로 휩쓸립니다. 주의: ン (n)과 혼동하지 마세요! ソ는 획이 위에서 아래로 가고 곡선이 더 수직적입니다. 외래어: ソファー (소파아) = 소파 / 소파.
주의 구간: 혼동하기 쉬운 가타카나

이 쌍은 모든 초보자를 당혹스럽게 합니다. 주의 깊게 학습하세요:

シ (shi) vs ツ (tsu): シ는 점이 대각선 위로 (왼쪽 아래에서 오른쪽 위로) 가고 곡선이 수평적입니다. ツ는 점이 대각선 아래로 (왼쪽 위에서 오른쪽 아래로) 가고 곡선이 수직적입니다. シ는 웃는 얼굴(수평 입), ツ는 눈물이 떨어지는 모양(수직 물방울)으로 기억하세요.

ソ (so) vs ン (n): 같은 원리입니다! ソ의 획은 아래로 휩쓸립니다 (ツ의 패턴과 같습니다). ン의 획은 위로 휩쓸립니다 (シ의 패턴과 같습니다). ソ = 짧고 날카롭습니다. ン = 길고 위로 올라갑니다.

ウ (u) vs ワ (wa) vs フ (fu): ウ는 윗획이 평평합니다. ワ는 위가 열려 있습니다. フ는 옆획 없이 두 획만 있습니다.

타행 -- タ チ ツ テ ト

タ
ta -- 히라가나 짝: た
한국어: 다/타
기억법: 큰 알파벳 "T"에 장식을 더한 모양입니다. 외래어: タクシー (타쿠시이) = 택시 / 택시.
チ
chi -- 히라가나 짝: ち
한국어: 치
기억법: 날카롭게 발차기하는 치어리더 -- ち의 각진 버전입니다. 외래어: チキン (치킨) = 치킨 / 치킨.
ツ
tsu -- 히라가나 짝: つ
한국어: 쓰 (가장 근사)
기억법: 쓰나미의 물방울 세 개 -- 점이 아래로 튀고 곡선이 수직으로 휩쓸립니다. 기억하세요: 점이 위에서 아래로 갑니다 (シ와 반대). 외래어: ツアー (츠아아) = 투어 / 투어.
テ
te -- 히라가나 짝: て
한국어: 데/테
기억법: 테레비전 전신주 -- 위에 가로 막대가 있고 아래로 직선이 내려갑니다. 외래어: テレビ (테레비) = TV / 텔레비전.
ト
to -- 히라가나 짝: と
한국어: 도/토
기억법: 토템 기둥 -- 하나의 세로선에 작은 가로 가지가 있습니다. 가장 단순한 가타카나입니다! 외래어: トイレ (토이레) = 화장실 / 화장실.

나행 -- ナ ニ ヌ ネ ノ

ナ
na -- 히라가나 짝: な
한국어: 나
기억법: 비스듬히 박힌 나사못 -- 대각선 획이 있는 십자 모양입니다. 외래어: ナイフ (나이후) = 나이프 / 나이프.
ニ
ni -- 히라가나 짝: に
한국어: 니
기억법: 두 개의 가로선 = ni (일본어로 2)! 글자 자체가 숫자 2처럼 보입니다. 외래어: ニュース (뉴우스) = 뉴스 / 뉴스.
ヌ
nu -- 히라가나 짝: ぬ
한국어: 누
기억법: 각진 그릇 안의 누들(면) -- 각진 교차 획입니다. 외래어: カヌー (카누우) = 카누 / 카누.
ネ
ne -- 히라가나 짝: ね
한국어: 네
기억법: 네트(그물) -- 그물처럼 교차하는 각진 획입니다. 외래어: ネクタイ (네쿠타이) = 넥타이 / 넥타이.
ノ
no -- 히라가나 짝: の
한국어: 노
기억법: 하나의 사선 -- "노!" 하고 날카롭게 긋는 느낌입니다. 가장 단순한 가타카나: 대각선 한 획뿐입니다. 외래어: ノート (노오토) = 노트 / 노트.

하행 -- ハ ヒ フ ヘ ホ

ハ
ha -- 히라가나 짝: は
한국어: 하
기억법: "하!" -- 두 획이 웃으며 팔을 벌린 모양으로 펼쳐집니다. 외래어: ハンバーガー (한바아가아) = 햄버거 / 햄버거.
ヒ
hi -- 히라가나 짝: ひ
한국어: 히
기억법: "히" 하고 웃는 사람 -- 옆으로 돌린 웃는 입처럼 보입니다. 외래어: ヒーロー (히이로오) = 히어로 / 히어로.
フ
fu -- 히라가나 짝: ふ
한국어: 후 (입술이 닿지 않음)
기억법: 후지산의 봉우리 -- 두 획이 단순한 산봉우리를 이룹니다. 외래어: フライドポテト (후라이도포테토) = 감자튀김 / 감자튀김.
ヘ
he -- 히라가나 짝: へ
한국어: 헤
기억법: 히라가나 へ와 똑같이 생겼습니다! 히라가나 짝과 동일한 몇 안 되는 가타카나 중 하나입니다. 단순한 산봉우리/지붕 모양입니다. 외래어: ヘリコプター (헤리코푸타아) = 헬리콥터 / 헬리콥터.
ホ
ho -- 히라가나 짝: ほ
한국어: 호
기억법: 신성한 십자가 -- 세로 획에 가로 막대와 비스듬한 양옆이 있습니다. 외래어: ホテル (호테루) = 호텔 / 호텔.

마행 -- マ ミ ム メ モ

マ
ma -- 히라가나 짝: ま
한국어: 마
기억법: 마름모 각도 -- 두 획이 부등호 모양의 날카로운 각을 이룹니다. 외래어: マクドナルド (마쿠도나루도) = 맥도날드 / 맥도날드.
ミ
mi -- 히라가나 짝: み
한국어: 미
기억법: 세 개의 가로선 = mi (일본어로 3)! ニ (ni=2)처럼 선이 세 개입니다. 외래어: ミルク (미루쿠) = 밀크 / 밀크.
ム
mu -- 히라가나 짝: む
한국어: 무
기억법: 소의 얼굴 -- 각진 선이 "무~" 하는 소를 만듭니다. 외래어: ムード (무우도) = 무드 / 무드.
メ
me -- 히라가나 짝: め
한국어: 메
기억법: "X" 표시 -- 메뉴에서 체크하는 것처럼 보입니다. 외래어: メニュー (메뉴우) = 메뉴 / 메뉴.
モ
mo -- 히라가나 짝: も
한국어: 모
기억법: 더 많은 선 -- 가로 막대들을 세로선이 연결하며, 선반을 더 추가하는 모양입니다. 외래어: モデル (모데루) = 모델 / 모델.

야행 -- ヤ ユ ヨ (3개만)

ヤ
ya -- 히라가나 짝: や
한국어: 야
기억법: 야크트(요트)의 돛대 -- 범선 장비처럼 각진 획입니다. 외래어: ヤクルト (야쿠루토) = 야쿠르트 / 야쿠르트.
ユ
yu -- 히라가나 짝: ゆ
한국어: 유
기억법: 유니크한 갈고리 -- 낚시 바늘에 막대가 하나 더 있는 모양입니다. 외래어: ユニフォーム (유니포오무) = 유니폼 / 유니폼.
ヨ
yo -- 히라가나 짝: よ
한국어: 요
기억법: 요가 자세 -- 알파벳 "E"에서 아래쪽 선이 없는 모양, 사람이 몸을 굽히는 형태입니다. 외래어: ヨーグルト (요오구루토) = 요거트 / 요거트.

라행 -- ラ リ ル レ ロ

히라가나와 같은 탄음 "r" -- 모음 사이의 한국어 ㄹ과 비슷합니다.

ラ
ra -- 히라가나 짝: ら
한국어: 라
기억법: 라비트(토끼) 귀 -- 위로 솟은 각진 획입니다. 외래어: ラーメン (라아멘) = 라면 / 라면.
リ
ri -- 히라가나 짝: り
한국어: 리
기억법: 두 개의 리버(강) -- り와 비슷하지만 더 날카롭습니다. い와 비슷하지만 오른쪽 획이 휘어집니다. 외래어: リモコン (리모콘) = 리모컨 / 리모컨.
ル
ru -- 히라가나 짝: る
한국어: 루
기억법: 나무 루트(뿌리) -- 두 선이 아래에서 뿌리처럼 펼쳐집니다. 외래어: ルール (루우루) = 룰 / 룰.
レ
re -- 히라가나 짝: れ
한국어: 레
기억법: 날카로운 레이스 바톤(릴레이 바톤) -- 위로 올라가며 꺾이는 하나의 각진 획입니다. 외래어: レストラン (레스토란) = 레스토랑 / 레스토랑.
ロ
ro -- 히라가나 짝: ろ
한국어: 로
기억법: 로봇의 입 -- 단순한 열린 직사각형입니다. 가장 기억하기 쉬운 가타카나입니다! 외래어: ロボット (로봇토) = 로봇 / 로봇.

와행 -- ワ ヲ + 특수: ン

ワ
wa -- 히라가나 짝: わ
한국어: 와
기억법: 위가 열린 지갑 -- ウ에서 윗획이 없는 모양입니다. 외래어: ワイン (와인) = 와인 / 와인.
ヲ
wo (발음은 "o") -- 히라가나 짝: を
한국어: 목적격 조사 을/를
기억법: 현대 일본어에서는 거의 사용되지 않습니다 (대신 を를 사용합니다). 아주 오래된 문헌이나 특수한 맥락에서만 볼 수 있습니다. フ 위에 가로 획이 하나 더 있는 모양입니다.
ン
n (독립 자음) -- 히라가나 짝: ん
한국어: ㄴ받침
기억법: 유일한 자음 단독 문자입니다. 주의: ソ (so)와 혼동하지 마세요! ン의 획은 위로 휩쓸립니다 (シ처럼), ソ의 획은 아래로 갑니다 (ツ처럼). 외래어: パン (판) = 빵 / 빵.
